# mockmania

simple python mocking of http api, able to catch http request and directly modify the output.

## step by step

### Create venv
$ python3 -m venv venv

### Activate venv
$ source venv/bin/activate

### Run the script
$ python main.py &

### Run the sample use
$ python sample_use.py
